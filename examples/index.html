app-store-badges — vanilla showcase

Pre-publish eyeball page. Top half renders badges on white; bottom half on black. Same locale matrix on both so region-distinct assets (pt-BR vs pt-PT, fr vs fr-CA, sl vs sl-SI) can be compared side-by-side.

Vanilla usage: import '@rewilok/app-store-badges/elements', then drop <app-store-badge> / <google-play-badge> anywhere.

On white

App Store with theme="black" (default) and Google Play (ships only black). Caption shows the requested locale; if the requested key isn't shipped for a product, the matcher resolves to the closest available — that's expected behavior, eyeball it here.
